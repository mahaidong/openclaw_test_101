## 使用说明

### 快速开始
1. 打开链接: https://mahaidong.github.io/openclaw_test_101/pong.html
2. 选择游戏模式 (双人对战 / 对战AI)
3. 点击"开始游戏"或按空格键
4. 使用控制方式游戏

### 控制说明
```
玩家1 (左侧):
  - 键盘: W (上), S (下)
  - 鼠标: 在左侧区域移动
  - 触摸: 触摸屏幕左侧

玩家2 (右侧):
  - 键盘: I (上), K (下)
  - 鼠标: 在右侧区域移动
  - 触摸: 触摸屏幕右侧

通用控制:
  - 空格键: 开始/暂停游戏
  - 按钮: 游戏界面上的控制按钮
```

## 故障排除

### 常见问题
1. **游戏无法开始**
   - 检查浏览器是否支持Canvas
   - 尝试刷新页面
   - 确保JavaScript已启用

2. **控制不响应**
   - 检查键盘输入焦点
   - 移动设备确保触摸支持
   - 尝试点击游戏区域获取焦点

3. **性能问题**
   - 关闭其他标签页释放资源
   - 更新浏览器到最新版本
   - 降低浏览器硬件加速设置
